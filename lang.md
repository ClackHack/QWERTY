
C++ language
QWERTY
Only letters can be used
State can be changed between binary, char, and int

o - output current cell
i- input current cell

c- char mode
b-bin mode
n - integer mode

E - begin loop
(If followed by a ‘q’ run till cell is 0
If followed by a ‘d’ run the cell’s value # times
L - close loop

A- move left
D - move right
W - move up
S - move down 

v - decrement 
y - increment
